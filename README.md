# 🛍️ Retail Sales Analysis using SQL Server

## 📌 Project Overview
This project focuses on analyzing retail sales data using SQL Server.  
It includes data exploration, data cleaning, and business insights using SQL queries such as aggregation, filtering, grouping, and window functions.

---

## 🎯 Objectives
- Understand dataset structure
- Perform data cleaning (handling null values)
- Explore customer behavior
- Analyze sales performance by category, gender, and time
- Identify top customers and best-selling periods

---

## 🧹 Data Cleaning
- Checked for missing values across all columns
- Removed records with NULL values to ensure data quality

---

## 📊 Key Analysis Performed

### 1. Dataset Overview
- Total number of records
- Unique customers
- Unique product categories

### 2. Sales Analysis
- Total sales per category
- Transactions with high sales (>1000)
- Top 5 customers based on total spending

### 3. Customer Insights
- Average age of customers per category
- Gender-based transaction analysis

### 4. Time-Based Analysis
- Sales by date filtering
- Monthly average sales
- Best performing month per year using window functions
- Orders distribution by shift (Morning, Afternoon, Evening)

---

## 🧠 Techniques Used
- SQL Aggregations (SUM, AVG, COUNT)
- GROUP BY / HAVING
- Window Functions (RANK)
- CTEs (Common Table Expressions)
- CASE Statements
- Date Functions (YEAR, MONTH, DATEPART)

---

## 🛠️ Tools Used
- SQL Server
- SSMS (SQL Server Management Studio)

---

## 📈 Key Insights
- Identified top customers contributing to revenue
- Found peak sales periods and shifts
- Analyzed category-wise performance
- Detected customer behavior patterns
